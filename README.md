# Notes Sharing API

## 📌 Project Overview
A full-stack web application for uploading and sharing notes publicly or privately.The system supports uploading notes, downloading it, viewing own and others public notes.

---

## 🚀 Features

### Users
- Register 
- Login
- Notes upload
- Notes download
- Sharing notes publicly

### System
- JWT authorization
- Clean pagination

---

## Tech Stack

Backend
- Python
- FastAPI
- SQLAlchemy
- JWT
- OAuth

Frontend:
- React
- Tailwind CSS

Database:
- PostgreSQL

---

## Installation Guide:

Backend:
- pip install -r requirements.txt
- uvicorn app.main:app reload

Frontend:
- npm install
- npm run dev
